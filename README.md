# Growatt Product Center V2.0 — Device Center

V2.0 marks the transition from a leadership demo to an incrementally maintained product platform.

## What is new

- Structured device master data for 12 device records
- Device taxonomy and category filtering
- Search by model, DTC, capability, region, or device type
- Click-through device detail pages
- Capability coverage, parameters, firmware, collectors, regions, and linked assets
- Existing V1.6 Handbook, compatibility, and PRD content retained

## Deploy

Upload all files to the GitHub Pages repository root and hard refresh after deployment.
