# Growatt Product Center V1.6 — PRD & Decision Center

V1.6 将 GPC 从“知识展示”继续推进到“产品决策与需求治理”。

## 本版新增

- PRD 与决策中心
- 登录地区与无电站异常排查 PRD
- 设备能力动态展示 PRD
- 旧版迁移与退场治理 PRD
- Approved / In Review / Draft 状态筛选
- 问题、决策、范围、排除项、验收标准结构化呈现
- 统一 PRD 模板
- 全局搜索增加 PRD 入口

## 部署

将仓库根目录文件上传到 GitHub Pages 使用的 `main / (root)`，等待 Actions 部署完成后强制刷新页面。
